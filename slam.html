BTS SIO - Option SLAM

Solutions Logicielles et Applications Métiers

Présentation de l'option SLAM

L'option SLAM (Solutions Logicielles et Applications Métiers) du BTS SIO forme des développeurs capables de concevoir, développer et maintenir des applications informatiques.

Compétences développées

  • Développement d'applications web et mobiles
  • Conception et gestion de bases de données
  • Programmation orientée objet
  • Développement front-end et back-end
  • Tests et déploiement d'applications
  • Maintenance et évolution des solutions logicielles

Technologies étudiées

HTML/CSS/JavaScript
PHP
Python
Java
SQL
Frameworks web

Débouchés professionnels

  • Développeur d'applications
  • Développeur web
  • Développeur mobile
  • Analyste programmeur
  • Technicien d'études informatiques
